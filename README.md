## Hola que tal!!👋
Valores basados en el compromiso y el orden. Durante mi trayectoria laboral he desarrollado la capacidad de
trabajar en equipo, contribuir con soluciones favorables al crecimiento empresarial y experiencia en el desarrollo
de software siguiendo metodologías ágiles (Scrum). Entre mis objetivos busco contribuir para el logro de metas
generales y responsabilidades a mi cargo, después con esfuerzo, trabajo y dedicación obtener estabilidad laboral.

Colaboré en un proyecto que destacó en la creación de un cotizador digital en Insumos Tecnológicos
Platino implementando tecnología Java, el framework Spring así como los basados en este último:
Spring Boot, Spring Security y Spring data JPA para la persistencia de datos utilizando el sistema
gestor de bases de datos relacional MariaDB y el patrón de diseño por capas MVC lo que permitió
la separación de responsabilidades en la arquitectura del proyecto, mejorando el mantenimiento y
la escalabilidad del mismo.

Participe en la migración de un sistema con arquitectura monolítica diseñado para ejecutar ventas
utilizando una estrategia de refactorización y pasando de una versión JAVA 8 a una versión
JAVA 21 e implementando el Framework de Spring para convertir la arquitectura principal en
una arquitectura de microservicios lo que permitió maximizar la escalabilidad, mantenibilidad y
eficiencia.
